## 一、工具简介

### 1.1 这个工具能做什么？

AI辅助投稿分析工具可以帮助您：

- **自动分类**：将投稿内容自动归类到教育、劳动、社会、科技等类别
- **智能摘要**：提取投稿的核心内容，生成 3-5 句关键摘要
- **热度评估**：评估事件的影响力和传播潜力（1-10分）
- **报告导出**：一键导出 TXT、Word、PDF 格式的分析报告

### 1.2 适用场景

- 处理社区收到的投稿内容
- 快速筛选高热度事件
- 批量分析多条投稿
- 生成标准化分析报告

## 二、安装与启动

### 2.1 环境要求

- 操作系统：Windows 7 及以上
- Python：3.8 或更高版本（如未安装，请访问 https://www.python.org/downloads/ 下载）

### 2.2 启动步骤

#### 方法一：一键启动（推荐）

1. 找到项目文件夹中的 `start.bat` 文件
2. **双击运行** `start.bat`
3. 浏览器会自动打开 http://localhost:7860
4. 看到工具界面即启动成功

#### 方法二：命令行启动

1. 打开命令提示符（按 Win+R，输入 `cmd`，回车）
2. 输入以下命令：
   ```
   cd e:\项目\ai-submission-analyzer
   python app.py
   ```
3. 浏览器访问 http://localhost:7860

### 2.3 首次使用配置

首次使用需要配置 API 密钥：

1. 点击界面右上角的 **"⚙️ 设置"** 标签
2. 在 **"🔑 API密钥"** 区域选择一个 AI 模型
3. 填入对应的 API Key（获取方式见下文）
4. 点击 **"💾 保存"** 按钮

## 四、功能使用指南

### 4.1 单个投稿分析

![首页 - 投稿分析界面](picture/首页.png)

**操作步骤：**

1. **粘贴内容**
   - 在左侧 **"📝 单个分析"** 文本框中粘贴投稿内容
   - 支持从微信、小红书、抖音、微博等平台复制的内容

2. **选择模型**
   - 在 **"🤖 AI模型"** 下拉框选择模型
   - 推荐选择 **DeepSeek**

3. **选择导出格式**
   - **TXT**：纯文本格式，通用性强
   - **Word**：适合编辑和打印
   - **PDF**：适合存档和分享
   - **两者都导出**：同时生成 TXT 和 Word
   - **三者都导出**：生成全部三种格式

4. **开始分析**
   - 点击 **"🚀 开始分析"** 按钮
   - 等待 3-10 秒（取决于内容长度）

5. **查看结果**
   - 右侧显示完整的分析报告
   - 包括：事件标题、分类、摘要、热度评分、影响评估

6. **下载报告**
   - 在报告下方点击对应的下载按钮

### 4.2 批量投稿分析

**适用场景：** 需要一次处理多条投稿

**操作步骤：**

1. **准备文件**
   - 创建一个 TXT 文件
   - 每行写一条投稿内容
   - 示例：
     ```
     某学校食堂卫生问题，学生集体腹泻...
     某公司强制员工加班，不支付加班费...
     某小区物业乱收费，业主维权困难...
     ```

2. **上传文件**
   - 点击 **"📦 批量处理"** 标签
   - 点击 **"上传文件"** 选择准备好的 TXT 或 CSV 文件

3. **开始分析**
   - 选择 AI 模型和导出格式
   - 点击 **"🚀 开始批量分析"**

4. **查看结果**
   - 左侧表格显示所有投稿的摘要信息
   - **点击任意行**，右侧显示该条投稿的详细分析

### 4.3 查看历史记录

![历史记录界面](picture/历史记录.png)

**操作步骤：**

1. 点击 **"📜 历史记录"** 标签
2. 可以看到所有分析过的投稿
3. 使用筛选功能：
   - 按 **分类** 筛选
   - 按 **热度** 筛选
4. 点击任意记录可重新查看详细分析

### 4.4 数据统计

![数据分析界面](picture/数据分析.png)

**操作步骤：**

1. 点击 **"📊 数据统计"** 标签
2. 查看：
   - **分类分布饼图**：各类别投稿占比
   - **7天趋势图**：近期投稿数量变化
   - **统计数据**：总数、平均热度等

### 4.5 自定义设置

![设置页面](picture/设置.png)

**修改分类类别：**

1. 点击 **"⚙️ 设置"** 标签
2. 在 **"📂 分类管理"** 区域：
   - 输入新类别名称，点击 **"➕ 添加"**
   - 点击已有类别旁的 **"🗑️"** 可删除

## 五、分析结果解读

### 5.1 事件分类

| 分类 | 包含内容 |
|------|---------|
| **教育** | 学校教育、教学改革、教育公平、考试招生、师德师风 |
| **劳动** | 劳动者权益、工资薪酬、加班文化、劳动纠纷、职场歧视 |
| **校园** | 校园生活、学生活动、校园安全、宿舍食堂、学生组织 |
| **社会** | 社会民生、公共政策、社区治理、社会公平、弱势群体 |
| **科技** | 科技发展、数据隐私、人工智能、互联网平台、网络安全 |
| **环境** | 环境污染、生态保护、气候变化、资源浪费 |
| **医疗** | 医疗资源、医患关系、公共卫生、医疗费用 |
| **其他** | 不属于以上类别的事件 |

### 5.2 热度评分

| 分数 | 等级 | 建议处理方式 |
|------|------|-------------|
| **8-10 分** | 🔴 高热度 | 优先关注，可能需要快速响应 |
| **5-7 分** | 🟡 中热度 | 正常关注，按流程处理 |
| **1-4 分** | 🟢 低热度 | 常规处理，可延后 |

### 5.3 评估维度说明

AI 会从以下维度评估事件：

- **影响范围**：受影响人群规模（个人/群体/社会层面）
- **传播潜力**：在社交媒体上的传播可能性
- **历史相似**：是否有类似先例事件
- **情感强度**：引发公众情绪共鸣的程度

## 六、常见问题解答

### Q1：启动后浏览器没有自动打开？

**解决方法：**
- 手动打开浏览器
- 在地址栏输入 `http://localhost:7860`
- 按回车访问

### Q2：提示"未配置 API 密钥"？

**解决方法：**
1. 确认已在设置页面保存 API 密钥
2. 检查密钥是否正确（以 `sk-` 开头）
3. 确认密钥没有过期或余额不足

### Q3：分析结果不准确？

**解决方法：**
- 尝试切换不同的 AI 模型
- 确保投稿内容完整、清晰
- 内容越长，分析越准确

### Q4：批量分析失败？

**解决方法：**
- 确保文件编码为 UTF-8
- 每行只写一条投稿
- 文件不要太大（建议不超过 100 条）

### Q5：Word 文件打开乱码？

**解决方法：**
- 使用 Microsoft Word 或 WPS 打开
- 不要用记事本打开 `.docx` 文件

### Q6：端口被占用？

**解决方法：**
- 关闭其他使用 7860 端口的程序
- 或重新运行 `start.bat`（会自动释放端口）

## 七、使用技巧

### 7.1 提高分析准确度

- **提供完整内容**：尽量粘贴完整的投稿内容
- **包含关键信息**：时间、地点、人物、事件经过
- **避免过于简短**：内容不少于 50 字效果更好

### 7.2 提高工作效率

- **批量处理**：收集多条投稿后一次性批量分析
- **使用历史记录**：之前分析过的内容可在历史记录中查找
- **自定义分类**：根据社区实际需求调整分类类别

### 7.3 报告使用建议

- **TXT 格式**：适合快速查看和复制粘贴
- **Word 格式**：适合进一步编辑和完善
- **PDF 格式**：适合存档和正式提交
