# AI辅助投稿分析工具 - 使用说明

## 📖 工具简介

本工具帮助社区志愿者快速处理投稿：自动分类、智能摘要、热度评估，减少人工看帖时间，让社区更快响应热点事件。

---

## 🚀 快速开始

### 第一步：安装环境

1. 确保电脑已安装 **Python 3.8+**
2. 打开命令行（Win+R 输入 cmd），执行：
```bash
cd 项目文件夹路径
pip install -r requirements.txt
```

### 第二步：配置AI模型密钥

选择一个免费AI模型，设置环境变量：

| 模型 | 推荐度 | 免费额度 | 环境变量 |
|------|--------|---------|---------|
| DeepSeek | ⭐⭐⭐⭐⭐ | 免费额度高 | `set DEEPSEEK_API_KEY=你的密钥` |
| 通义千问 | ⭐⭐⭐⭐ | 免费额度高 | `set DASHSCOPE_API_KEY=你的密钥` |
| 智谱GLM | ⭐⭐⭐ | 有免费额度 | `set GLM_API_KEY=你的密钥` |
| Kimi | ⭐⭐⭐ | 有免费额度 | `set MOONSHOT_API_KEY=你的密钥` |

**获取密钥方法：**
- DeepSeek：访问 https://platform.deepseek.com 注册 → API Keys → 创建
- 通义千问：访问 https://dashscope.console.aliyun.com 开通
- 智谱GLM：访问 https://open.bigmodel.cn 注册
- Kimi：访问 https://platform.moonshot.cn 注册

### 第三步：启动工具

```bash
python app.py
```

启动后浏览器自动打开 http://localhost:7860

## 📋 使用流程

### 方式一：直接粘贴投稿

1. 打开工具网页
2. 在左侧文本框粘贴投稿内容（支持从微信、小红书、抖音复制的内容）
3. 选择AI模型（推荐DeepSeek）
4. 选择导出格式（TXT / Word / 两者都导出）
5. 点击 **🚀 开始分析**
6. 右侧显示分析报告，下方可下载导出文件

### 方式二：使用油猴脚本采集

1. 安装 Tampermonkey 浏览器扩展
2. 将 `tampermonkey-collector.js` 内容新建为油猴脚本
3. 浏览微博/知乎/百度热搜等网页时，右下角出现 🔍 按钮
4. 点击按钮打开采集面板
5. 点击「提取当前页」或「提取热搜榜」
6. 点击「复制内容」→ 粘贴到分析工具
7. 或点击「发送到分析工具」自动跳转

## 📊 分析结果说明

### 事件分类

| 类别 | 说明 |
|------|------|
| 教育 | 学校教育、教学改革、教育公平、考试招生等 |
| 劳动 | 劳动者权益、工资薪酬、加班文化、劳动纠纷等 |
| 校园 | 校园生活、学生活动、校园安全、宿舍食堂等 |
| 社会 | 社会民生、公共政策、社区治理、社会公平等 |
| 科技 | 科技发展、数据隐私、人工智能、互联网平台等 |
| 环境 | 环境污染、生态保护、气候变化等 |
| 医疗 | 医疗资源、医患关系、公共卫生等 |
| 其他 | 不属于以上类别的事件 |

### 热度评分

| 分数 | 等级 | 含义 |
|------|------|------|
| 8-10 | 🔴 高热度 | 影响范围大、传播性强、需优先关注 |
| 5-7 | 🟡 中热度 | 有一定影响、值得关注 |
| 1-4 | 🟢 低热度 | 影响有限、可常规处理 |

### 评估维度

- **影响范围**：受影响人数多少（个人/群体/社会层面）
- **传播潜力**：事件是否容易在社交媒体传播
- **历史相似**：是否有类似先例事件
- **情感强度**：是否容易引发公众情绪共鸣

## 🌐 油猴脚本安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → 添加新脚本
3. 将 `tampermonkey-collector.js` 文件内容粘贴进去
4. 保存（Ctrl+S）
5. 访问任意网页，右下角出现 🔍 浮动按钮

### 油猴脚本支持平台

- 微博 (weibo.com)
- 知乎 (zhihu.com)
- 百度热搜 (top.baidu.com)
- 抖音 (douyin.com)
- 小红书 (xiaohongshu.com)
- B站 (bilibili.com)
- 任意网页（通用提取模式）

## ❓ 常见问题

**Q: 启动报错"未配置API密钥"**
A: 需要先设置环境变量。Windows打开cmd输入：`set DEEPSEEK_API_KEY=你的密钥`

**Q: 分析结果不准确**
A: 可尝试切换不同AI模型，不同模型对同一内容的分析可能有差异。

**Q: 油猴脚本无法提取内容**
A: 部分网站结构变化可能导致选择器失效，可手动复制内容粘贴到工具中。

**Q: Word导出乱码**
A: 请使用 Microsoft Word 或 WPS 打开，不要用记事本打开 .docx 文件。
