# AI辅助投稿分析工具 - 使用说明

## 📖 工具简介

本工具帮助社区志愿者快速处理投稿：自动分类、智能摘要、热度评估，减少人工看帖时间，让社区更快响应热点事件。

---

## 🚀 快速开始

### 第一步：安装环境

1. 确保电脑已安装 **Python 3.8+**
2. 打开命令行，执行：
```bash
cd 项目文件夹路径
pip install -r requirements.txt
```

### 第二步：配置AI模型密钥

选择一个AI模型，在设置页面填入API Key：

| 模型 | 推荐度 | 免费额度 |
|------|--------|---------|
| DeepSeek | ⭐⭐⭐⭐⭐ | 免费额度高 |
| 通义千问 | ⭐⭐⭐⭐ | 免费额度高 |
| 智谱GLM | ⭐⭐⭐ | 有免费额度 |
| Kimi | ⭐⭐⭐ | 有免费额度 |

获取密钥方法：
- DeepSeek：https://platform.deepseek.com
- 通义千问：https://dashscope.console.aliyun.com
- 智谱GLM：https://open.bigmodel.cn
- Kimi：https://platform.moonshot.cn

### 第三步：启动工具

```bash
python app.py
```

启动后浏览器自动打开 http://localhost:7860

## 📋 使用流程

### 单个投稿分析

1. 在左侧文本框粘贴投稿内容
2. 选择AI模型（推荐DeepSeek）
3. 选择导出格式（TXT / Word / PDF）
4. 点击 **🚀 开始分析**
5. 右侧显示分析报告，下方可下载导出文件

### 批量投稿分析

1. 切换到"📦 批量处理"标签
2. 上传 CSV 或 TXT 文件（每行一条投稿）
3. 选择模型和导出格式
4. 点击 **🚀 开始批量分析**
5. 点击表格行查看详细分析

### 油猴脚本采集

1. 安装 Tampermonkey 浏览器扩展
2. 将 `tampermonkey-collector.js` 新建为油猴脚本
3. 浏览微博/知乎/百度热搜时使用

## 📊 分析结果说明

### 事件分类

| 类别 | 说明 |
|------|------|
| 教育 | 学校教育、教学改革、教育公平、考试招生等 |
| 劳动 | 劳动者权益、工资薪酬、加班文化、劳动纠纷等 |
| 校园 | 校园生活、学生活动、校园安全、宿舍食堂等 |
| 社会 | 社会民生、公共政策、社区治理、社会公平等 |
| 科技 | 科技发展、数据隐私、人工智能、互联网平台等 |
| 环境 | 环境污染、生态保护、气候变化等 |
| 医疗 | 医疗资源、医患关系、公共卫生等 |
| 其他 | 不属于以上类别的事件 |

### 热度评分

| 分数 | 等级 | 含义 |
|------|------|------|
| 8-10 | 🔴 高热度 | 影响范围大、传播性强、需优先关注 |
| 5-7 | 🟡 中热度 | 有一定影响、值得关注 |
| 1-4 | 🟢 低热度 | 影响有限、可常规处理 |

## ❓ 常见问题

**启动报错"未配置API密钥"**
在"⚙️ 设置 → 🔑 API密钥"中填写Key。

**分析结果不准确**
切换不同AI模型重试。

**Word导出乱码**
用 Microsoft Word 或 WPS 打开。
