## 📦 项目分层架构

### 1. 表现层 (Presentation Layer)

**文件**: `app.py`, `tampermonkey-collector.js`

**职责**:
- 接收用户输入（文本粘贴 / 油猴脚本采集）
- 展示分析结果（Markdown渲染）
- 提供文件下载（TXT / Word）
- AI模型选择与配置

**关键接口**:
```
process_submission(text, export_format, provider) → (markdown_report, txt_file, word_file)
```

### 2. 业务逻辑层 (Business Logic Layer)

**文件**: `ai_engine.py`

**职责**:
- 编排AI调用流程（标题→分类→摘要→评估）
- 构造Prompt提示词
- 解析AI返回的JSON结果
- 异常处理与降级

**核心函数**:
| 函数 | 输入 | 输出 | 说明 |
|------|------|------|------|
| `generate_title(text)` | 事件文本 | 标题字符串 | 生成≤20字标题 |
| `classify(text)` | 事件文本 | `{category, confidence, reason}` | 8类分类 |
| `summarize(text)` | 事件文本 | 摘要字符串 | 3-5句关键信息 |
| `assess_impact(text)` | 事件文本 | `{heat_score, ...}` | 热度1-10 + 四维评估 |
| `analyze_submission(text)` | 事件文本 | 完整分析结果dict | 编排以上4步 |

### 3. AI模型服务层 (AI Service Layer)

**文件**: `config.py`

**职责**:
- 管理4种国产AI模型配置
- 统一OpenAI兼容接口
- 环境变量读取与密钥管理
- 分类类别定义

**模型配置**:
```python
API_CONFIGS = {
    "deepseek": { base_url: "https://api.deepseek.com", model: "deepseek-chat" },
    "qwen":     { base_url: "https://dashscope.aliyuncs.com/compatible-mode/v1", model: "qwen-turbo" },
    "glm":      { base_url: "https://open.bigmodel.cn/api/paas/v4", model: "glm-4-flash" },
    "kimi":     { base_url: "https://api.moonshot.cn/v1", model: "moonshot-v1-8k" },
}
```

### 4. 数据输出层 (Data Output Layer)

**文件**: `report_generator.py`

**职责**:
- 生成Markdown格式报告（页面展示）
- 导出TXT纯文本文件
- 导出Word文档（含表格、颜色标注）

**输出格式**:
- **Markdown**: 页面实时展示，含表格、emoji标注
- **TXT**: 纯文本，兼容性好
- **Word**: 专业排版，热度分颜色标注（红/橙/绿），表格样式

## 🔄 数据流图

```
用户输入                    AI处理                      输出
─────── ───────────────── ──────── ──────────────── ────────

粘贴文本 ──→ process_submission() ──→ Markdown报告 ──→ 页面展示
   │              │
   │              ├──→ generate_title()  ──→ 事件标题
   │              ├──→ classify()        ──→ 类别+置信度
   │              ├──→ summarize()       ──→ 3-5句摘要
   │              └──→ assess_impact()   ──→ 热度分+四维评估
   │                                          │
   │                              ┌───────────┼───────────┐
   │                              ▼           ▼           ▼
   │                          export_txt()  export_word()  页面展示
   │                              │           │
   └──────────────────────────────┴───────────┘
                                  ▼
                              output/ 目录
                           ├── 投稿分析_xxx.txt
                           └── 投稿分析_xxx.docx


油猴脚本采集流程:
─────────────────

网页内容 ──→ extractCurrentPage() ──→ buildEventDescription() ──→ 复制/发送
   │
   ├── 微博/知乎/百度热搜等 → 定制选择器提取
   └── 通用网页 → body文本智能提取
```
